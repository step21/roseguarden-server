Welcome to RoseGuarden

This user interface will guide you through the setup of your RoseGuarden-node. It will be assumed, that you have already setup your host system carefully. If you are not sure whether all preparations are made correctly, please double-check. You will find informations and a helping hand on the projects github-page.

Please recheck, that your host system have a static IP-address.
A changing IP (e.g. DHCP) could lead to malfunction or data-loss.
